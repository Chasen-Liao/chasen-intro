About
me.

我是 Chasen，广东财经大学计算机科学在读。Claude Code 发烧友、vibecoding 资深玩家——几乎每天都在用 AI 把想法 build 成新东西。Chasen 取自「chase」：不断追逐，无限进步。

SuperMew 项目里我手写了 Dense+BM25 Hybrid 检索 + RRF 融合、三级分块 Auto-merging、三层记忆架构，并用 CMRC2018/2019 做了消融评测——结论是 RRF 已足够强，HyDE 反而引入偏差。Pi Agent Desktop 则是用 Electron + React 做出个人极简版 Codex。

目前正在寻找AI 应用开发 / 前端实习机会，愿意从 Agent 工程链路、RAG 系统、前端工程化到桌面端全链路投入。
