About
me.

我是 Chasen，广东财经大学计算机科学在读。Claude Code 发烧友、vibecoding 资深玩家——几乎每天都在用 AI 把想法 build 成新东西。Chasen 取自「chase」：不断追逐，无限进步。

能力上，我擅长 AI Agent / RAG 系统的工程化落地——从 Hybrid 检索、记忆架构，到前端工程化与桌面客户端，习惯一个人打通从想法到上线的全链路。兴趣在 AI 的边际与商业应用：日常用 OpenDesign 出设计、Claude Code 写代码，配合 LangGraph、React/TS 把一句想法快速 build 成上线产品，也爱折腾工具链、写技术随笔，把每次探索都沉淀下来。

目前正在寻找AI Agent 工程师 | AIGC 工程师方向的机会，愿意在 Agent 工程链路、RAG 系统、AIGC 应用到前端工程化全链路投入。

我是 Chasen，长期折腾 Claude Code、Agentic RAG、桌面客户端与个人效率工具。我的内容围绕一个问题：普通开发者如何用 AI 把想法更快地变成真实产品。

我会记录从 prompt、设计、检索、记忆架构到前端交互的完整过程，也会把踩坑、评测、工具链和可复用方法写成随笔或开源项目。

这里不是教程集合，而是一个持续迭代的 build log：用真实项目验证 AI 的边界，再把经验沉淀给下一次构建。
